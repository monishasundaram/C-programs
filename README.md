# C Programming – Level 1 Basic Programs

## 📌 Overview

This repository contains Level 1 basic C programs designed for beginners. These programs help build a strong foundation in:

* Input and Output
* Variables and Data Types
* Operators
* Conditional Statements (if-else)
* Basic Loops

All programs are written using standard C and can be compiled using GCC or any C compiler.

---

## 🛠 Requirements

* GCC Compiler (or any C compiler)
* VS Code / CodeBlocks / Turbo C (any IDE)

To compile using GCC:

```bash
gcc filename.c -o output
./output
```

---

## 📂 List of Programs

### 1️⃣ Hello World

Prints "Hello World" on the screen.

### 2️⃣ Sum of Two Numbers

Takes two integers as input and prints their sum.

### 3️⃣ Area of Circle

Calculates area using formula:

Area = π × r²

### 4️⃣ Swap Two Numbers

Swaps two numbers without using a third variable.

### 5️⃣ Even or Odd

Checks whether a number is even or odd using modulus operator.

### 6️⃣ Largest of Three Numbers

Finds the largest number using conditional statements.

### 7️⃣ Positive, Negative or Zero

Determines whether a number is positive, negative, or zero.

### 8️⃣ Celsius to Fahrenheit

Converts temperature using formula:

F = (C × 9/5) + 32

### 9️⃣ Multiplication Table

Prints multiplication table of a given number using a loop.

### 🔟 Simple Interest

Calculates Simple Interest using formula:

SI = (P × R × T) / 100

* How to declare and use variables
* Basic arithmetic operations
* Conditional logic (if-else)
* Simple loop usage
